# I Have Money - Minimal Cozy Menu Icons

ไฟล์ SVG แยกสำหรับใช้ในแอป `i-have-money-app`

## Files
- หน้าแรก = `menu-home.svg`
- เพิ่ม = `menu-add.svg`
- งบ = `menu-budget.svg`
- กราฟ = `menu-chart.svg`
- ตั้งค่า = `menu-settings.svg`
- ปฏิทิน = `menu-calendar.svg`
- รายรับ = `menu-income.svg`
- รายจ่าย = `menu-expense.svg`
- ซิงก์/สำรอง = `menu-sync.svg`
- โหมดมืด = `menu-dark.svg`
- แมว = `menu-cat.svg`
- นก = `menu-bird.svg`

## Usage
แนะนำวางไฟล์ไว้ที่:

```text
i-have-money-app/icons/menu/
```

แล้วอ้างอิงใน HTML/CSS/JS เช่น:

```html
<img src="icons/menu/menu-home.svg" alt="หน้าแรก" class="menu-icon">
```

```css
.menu-icon {
  width: 28px;
  height: 28px;
  object-fit: contain;
}
```
